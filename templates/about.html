About This Project

This application uses machine learning models like LSTM and XGBoost to predict stock prices based on historical data. It provides visualizations of technical indicators and forecasting results.

Built with Python, Flask, and Plotly, this tool aims to assist users in understanding stock trends and making informed decisions. Upload your stock data in CSV format and explore the predictions!
